## Box model 구성

- `Margin`  테두리 바깥의 외부 여백
- `Border`  테두리 영역
- `Padding`  테두리 안쪽의 내부 여백
- `Content`  요소의 실제 내용
- shorthand

```html
.margin-1{
	margin: 10px;					# 상하좌우
}
.margin-2{
	margin: 10px 20px;				# 상하 / 좌우
}
.margin-3{
	margin: 10px 20px 30px;			# 상 / 좌우 / 하
}
.margin-4{
	margin: 10px 20px 30px 40px;	# 상 / 우 / 하 / 좌 (시계 방향)
}
```
